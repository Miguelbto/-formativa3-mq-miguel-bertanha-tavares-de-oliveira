## Galaxy Tours 

#### Venha conhecer nosso site para explorara e visitar diferentes mundos em nossa galaxia.

### Tecnologias implementadas:

1. Temos customização no desing do site para tablets com até 768px de tamanho, assim deixamos duas colunas para a visualização do usuario.

2. Fizemos também um desing para celulares com até 480px, que ficaram com apenas 1 coluna para a visualização, deixamos o cabeçalho totalmente centralizado e retiramos a dica exclusiva do desktop.
